# AI-PLAT 用户手册

## 第二部分：核心功能操作指南

本部分将 AI-PLAT 的核心操作拆分为 7 个连续专题，覆盖项目准备、资源管理、标注、工作流、训练、发布与部署。可按实际任务直接进入对应专题，也可按推荐主线依次完成。

---

### 推荐操作主线

创建项目并添加成员 -> 获取算法组件 -> 上传数据和模型资源 -> 创建标签标注任务 -> 编排工作流 -> 训练与预测验证 -> 发布模型 -> 快速部署或生产部署。

### 七个专题说明

| 专题 | 适用任务 |
| --- | --- |
| 01 项目与组件准备 | 创建项目、分配成员和获取算法能力 |
| 02 数据与模型资源管理 | 上传、预览、版本化和下载数据或模型 |
| 03 数据标注与审核 | 创建标注任务并完成在线标注 |
| 04 工作流编排与运行 | 连接组件、资源并管理任务运行 |
| 05 模型训练与预测验证 | 训练模型、预测测试集并查看结果 |
| 06 模型发布与快速部署 | 发布模型，创建 Demo 或临时 API |
| 07 生产部署与运维管理 | 长期服务上线、监控、更新与回退 |

### 开始前检查

开始前建议确认已完成第一部分中的账号登录、项目权限和基础环境检查；如当前账号没有项目、组件或算力权限，请先联系项目管理员。
