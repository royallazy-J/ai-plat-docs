# AI-PLAT 用户手册

## 第三部分：场景教程与支持附录

本部分用于培训演示、交付复盘和日常支持。通过典型场景、常见问题、检查清单和支持方式，帮助用户完成核心操作后的验证与收尾。

---

### 推荐阅读顺序

按典型场景完成一次完整操作验证 -> 根据 FAQ 排查问题 -> 使用检查清单复核交付 -> 通过支持方式提交问题。

### 本部分包含

| 子章节 | 适用任务 |
| --- | --- |
| 01 典型场景操作教程 | 按真实业务链路完成训练、预测和部署验证 |
| 02 常见问题 FAQ | 快速定位账号、资源、工作流和部署问题 |
| 03 使用检查清单 | 交付或验收前逐项复核 |
| 04 联系支持方式 | 明确问题反馈所需信息和联系渠道 |
| 05 后续可补充内容 | 规划后续高级文档扩展 |
| 06 版本记录与手册结束 | 查看文档版本并完成阅读收尾 |

### 使用建议

遇到具体问题时可直接进入 FAQ；进行培训、交付或验收时，建议按典型场景和检查清单顺序阅读。
